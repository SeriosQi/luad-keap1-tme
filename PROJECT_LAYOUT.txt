LUAD KEAP1 TME — Project layout
================================

luad_keap1_tme/
├── .gitignore
├── PROJECT_LAYOUT.txt          ← this file
└── task1_bulk_predatory_matrix/
    ├── config.R                ← paths, gene sets, parameters
    ├── run_task1_pipeline.R    ← main entry (default: Scheme A Xena)
    ├── install_dependencies.R
    ├── R/                      ← analysis scripts (Step 1–5)
    │   ├── 01_download_tcga_xena.R   ← Scheme A: 2-file Xena download
    │   ├── 01_download_tcga.R        ← Scheme B: GDC (optional, --gdc)
    │   ├── 02_keap1_mutation_status.R
    │   ├── 03_differential_expression.R
    │   ├── 04_correlation_heatmap.R
    │   ├── 05_geo_validation.R
    │   └── utils.R
    ├── scripts/
    │   └── correlation_heatmap.py
    ├── data/
    │   ├── raw/xena/           ← downloaded .tsv.gz (not in git)
    │   ├── processed/          ← .rds caches, keap1_status.csv
    │   └── geo/                ← GEO validation caches
    ├── results/                ← analysis outputs (tables + figures)
    │   ├── *.csv               ← DEA, correlation, module scores
    │   └── figures/            ← PDF/PNG plots
    └── logs/                   ← pipeline logs (not in git)
